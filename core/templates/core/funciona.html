Aplicación web para el intercambio y venta de libros, el website se crea con la finalidad de incentivar la lectura y elevar el conocimiento, ahorrando dinero y evitando que se deterioren libros que no se usan, para que otras personas les saquen provecho.

Donar

Vender

Esta plataforma te ofrece la opción de vender algunos de tus libros favoritos, comunicarte con los usuarios que esten interesado en el libro que ofreces y llegar a un acuerdo de venta y establecer el metodo de envío.

Intercambiar
